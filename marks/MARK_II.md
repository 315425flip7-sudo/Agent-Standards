# BIBLE / Mark II: The First Proper Chapter

## Objective
Establish the conceptual and procedural "Soul" and "Mind" that precedes the "Body". This Mark codifies the refined frameworks for all future Hermes agents.

## I. Refined Framework: The 5 Pillars (The Shape)
The 5 Pillars define the **Anatomy** of a Hermes project or agent.

1. **Structure (Foundational Support):** Core code, dependencies, and project layout.
2. **Metabolism (Processing):** Input/Output cycles, resource lifecycle, and execution loops.
3. **Infrastructure (Defense & Transportation):** Security, networking, resource distribution, and error handling.
4. **Lineage (Reproduction):** The "Bible", AARs, skill generation, and historical continuity.
5. **Management (Coordination):** High-level logic, state management, and decision-making flow.

## II. The Trinity (The Lens)
Every component of the Agent-Standards must be viewed through:
- **Soul (Why):** The purpose and intent.
- **Mind (How):** The procedure and logic.
- **Body (What):** The actual implementation and artifacts.

## III. Protocol Baseline
This Mark enforces the **3-Turn Operational Protocol** (PLAN/EXECUTE/DEBRIEF) as the standard for all Project Mode operations.
